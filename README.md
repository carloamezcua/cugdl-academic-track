## Materias

### Arquitectura de Sistemas de Inteligencia Artificial

Introducción a los fundamentos de arquitecturas de sistemas orientados a inteligencia artificial.

| Archivo | Descripción |
|---------|-------------|
| `hola.py` | Primer ejercicio: Hola Mundo en Python |

### Machine Learning y Procesamiento de Datos

Exploración de técnicas de machine learning y análisis de datos con Python, incluyendo regresión lineal y clasificación.

#### Notebooks

| Notebook | Tema |
|----------|------|
| `01-26_regresion_lineal.ipynb` | Regresión lineal |
| `02-09_titanic_v0.ipynb` | Clasificación con dataset Titanic (v0) |
| `02-16_titanic_v1.ipynb` | Clasificación con dataset Titanic (v1) |

#### Datasets

| Archivo | Descripción |
|---------|-------------|
| `estudio_vs_calificacione.csv` | Relación entre horas de estudio y calificaciones |
| `titanic_train.csv` | Dataset de entrenamiento del Titanic |
